**How to Use the Ledger**

* Link the ledger directly inside decks or proposals to provide a timeline slide without reformatting the data.
* The "Linked Artefacts" column references filenames or sections inside this repository, making it easy to cross-link reports, readiness checklist items, and governance decisions.
* When adding new wells, replicate the markdown table format so the documentation registry renderer can parse it consistently.
