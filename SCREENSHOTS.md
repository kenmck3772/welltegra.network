## üì± Tools & Setup

### Recommended: Chrome DevTools

**Best for consistent screenshots across devices:**

1. **Open DevTools:**
   - Windows/Linux: `F12` or `Ctrl+Shift+I`
   - Mac: `Cmd+Option+I`

2. **Toggle Device Mode:**
   - Windows/Linux: `Ctrl+Shift+M`
   - Mac: `Cmd+Shift+M`

3. **Select Device:**
   - iPhone 14 Pro (390x844)
   - iPad Pro (1024x1366)
   - Desktop (1920x1080)

4. **Capture Screenshot:**
   - Click 3-dot menu in DevTools
   - "Capture screenshot" (visible area)
   - "Capture full size screenshot" (entire page)

---

### Alternative: Browser Extensions

**Awesome Screenshot (Chrome/Edge):**
- [Install from Chrome Web Store](https://chrome.google.com/webstore/detail/awesome-screenshot)
- Capture full page
- Annotate (add arrows, highlights)
- Export as PNG

**FireShot (Firefox):**
- [Install from Firefox Add-ons](https://addons.mozilla.org/en-US/firefox/addon/fireshot/)
- Capture entire page
- Save as PNG

---

## üé® Screenshot Best Practices

### Image Quality
- **Resolution:** Minimum 1920x1080 for desktop
- **Format:** PNG (not JPG - preserves text sharpness)
- **File Size:** Compress if > 500 KB (use [TinyPNG.com](https://tinypng.com/))

### Composition
- **Avoid clutter:** Close unnecessary browser tabs
- **Hide bookmarks bar:** Press `Ctrl+Shift+B` (Chrome)
- **Full page:** Capture entire page, not just viewport
- **Highlight interactions:** Show expanded/hover states

### Annotation (Optional)
- Add arrows pointing to key features
- Highlight interactive elements (buttons, filters)
- Add captions for complex features
