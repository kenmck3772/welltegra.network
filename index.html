Well‑Tegra Demo Bundle

This ZIP loads instantly. Replace the CDN with a local Tailwind build when you’re ready for production.

How to use

  1. Open index.html in a browser.
  2. When ready for production, build assets/tailwind.css and swap the CDN script for a stylesheet link.
  3. Copy your existing images/videos into /assets and change URLs to assets/… if you want full offline mode.
